Код рабтает так же как и в модуле. Для работы из браузера прошу перейти по ссылке http://localhost:8080/albums/***
где *** - имя артиста.
для боавления нового артиста в cmd прошу ввести 
-f POST http://localhost:8080/albums artist="***" genre="***" album="***" year="***"
где *** необходимо заполнить информацией по Вашему выбору

также не забудьте перед запуском сервера перейти с помощью команды "cd" в директорию, где Вы сохранили скачанные файлы.

Желаю здоровья :)
